## Dust library {{ anchor: false }}

The Dust Developer Platform is a technical documentation designed for developers and data scientists who want to build and
deploy Large Language Model (LLM) apps executed in a secure and scalable environment.

**For best practices and examples on how to use Dust or LLM prompting in general, check out [the Dust Library](https://dust-tt.notion.site/Library-1aa97619d56d4d9f9c21d7ce89d23dbb).**

---

## Getting started {{ anchor: false }}

To get started, you can learn more about the basic concepts around [Dust
Apps](/introduction) or get an [Overview](/overview) of the Dust platform. You can then follow our
[Quickstart](/quickstart) guide to build and run your first Dust App.

## Dive in {{ anchor: false }}

Dive into the details of building a Dust app by reviewing documentation for the [Core
Blocks](/core-blocks) and [Integration Blocks](/integration-blocks). Learn how to call a Dust app by
API by reviewing documentation for the [Runs](/runs) API resource. Discover how [Data
Sources](/data-sources) can be used to provide extended context to your Dust apps with our
[Document Q&A](/guide-document-qa) guide.

## Support {{ anchor: false }}

We're available on [Discord](https://discord.gg/8NJR3zQU5X) and
[GitHub](https://github.com/dust-tt/dust) to answer your questions and help you get started.
